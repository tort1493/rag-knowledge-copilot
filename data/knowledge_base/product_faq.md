# Product FAQ

Atlas Knowledge Copilot supports PDF, Markdown, and plain text documents for indexing. Spreadsheet ingestion is not available in the current release.

The default maximum document size for self-service uploads is 25 MB per file. Larger imports require support review.

Users can filter indexed content by department and date when those metadata fields are provided at upload time. Metadata filters are optional, not required.

The application provides source citations in answers so users can inspect the retrieved evidence directly.
